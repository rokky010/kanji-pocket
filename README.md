# 漢字ポケット

スマホ向けの漢字学習Webアプリです。フレームワークなし（HTML / CSS / JavaScriptのみ）で作られています。

## ファイル構成

```
kanji-pocket/
├── index.html   … 画面のHTML（ホーム / クイズ / 結果 / 一覧）
├── style.css    … デザイン（iPhone風・レスポンシブ）
├── data.js      … 問題データ（第5回〜第8回、全101問）
└── app.js       … アプリのロジック（出題・採点・保存）
```

## 使い方

1. ホーム画面で回（第5回〜第8回、またはすべての回）を選ぶ
2. 「この回をはじめる」をタップ
3. カタカナの部分を漢字で考えて「こたえを見る」をタップ
4. 〇（わかった）／△（あやしい）／✕（わからない）で自己採点
5. 最後まで解くと結果画面が表示される。「もう一度」で順番をシャッフルして再挑戦できる
6. ホーム画面の「まとめて見直す」から、いつでも回ごとの記録を見返せる

採点結果は端末の **localStorage** に保存されます（サーバー通信なし）。ブラウザのデータを消去すると記録も消えるのでご注意ください。

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作成する
2. このフォルダの中身（`index.html`, `style.css`, `data.js`, `app.js`）をリポジトリ直下にアップロードする
3. リポジトリの `Settings` → `Pages` を開く
4. `Source` を `Deploy from a branch` にし、ブランチを `main`（フォルダは `/root`）に設定して保存する
5. しばらくすると `https://ユーザー名.github.io/リポジトリ名/` でアプリが公開される

## 問題データについて

いただいた画像をもとに、以下の4回・計101問を収録しています（一部、判読が難しかった数問は除いています）。

- 第5回：ケンバン 〜 アテサキ（25問）
- 第6回：鉛筆のシン 〜 ヤセ（24問）
- 第7回：ソジョウ 〜 ソウソフ（26問）
- 第8回：サンロク 〜 キレツ（26問）

問題を追加・修正したい場合は `data.js` の配列に項目を追加するだけで反映されます。
